Clasificación de Sistemas Operativos.

Los sistemas operativos se pueden clasificar atendiendo a:

Administración de Tareas

  • Monotarea: Sistemas que permiten ejecutar un solo programa a la vez.
  • Multitarea: Sistemas que posibilitan la ejecución simultánea de varias tareas o programas.

Administración de Usuarios

  • Monousuario: Sistemas diseñados para ser utilizados por un único usuario, como en los ordenadores personales.
  • Multiusuario: Sistemas que permiten que varios usuarios trabajen y ejecuten sus programas al mismo tiempo.



Organización Interna o Estructura

  • Monolítico: El sistema operativo funciona como una única unidad donde todos los componentes están integrados.
  • Jerárquico: Organización en niveles que separa las funciones según su importancia o acceso.
  • Cliente-servidor: Modelo basado en la interacción entre clientes (que solicitan servicios) y servidores (que los proporcionan).



Manejo de Recursos o Acceso a Servicios

  • Centralizados: Sistemas que utilizan únicamente los recursos de un ordenador.
  • Distribuidos: Sistemas que permiten acceder y utilizar los recursos (CPU, memoria, periféricos, etc.) de múltiples ordenadores de manera simultánea.



Ejemplos de sistemas operativos

DOS

Familia de sistemas operativos para PC. Sus siglas significan Disk Operating System. Fue creado para ordenadores IBM y fue muy popular. Carece de interfaz gráfica y no es multiusuario ni multitarea. Con la aparición del sistema operativo Windows fue rápidamente sustituido.

WINDOWS

Familia de sistemas operativos no libres desarrollados por la empresa Microsoft Corporation, que se basan en una interfaz gráfica que se caracteriza por la utilización de ventanas, la última versión es Windows 11 es el resultado de la convergencia de dos líneas de desarrollo que hasta entonces se mantenían separadas en otras versiones de Windows: las versiones basadas en MS-DOS y las versiones basadas en NT (New Technology).

UNÍX

Familia de sistemas operativos que comparten unos criterios de diseño e interoperabilidad en común, que descienden de una primera implementación original de AT&T. Se trata de un sistema operativo portable, multitarea y multiusuario.

GNU/LINUX

Sistema operativo libre creado por Richard Stallman. GNU es un acrónimo que significa GNU no es Unix («GNU's Not Unix»). Un sistema operativo libre quiere decir que los códigos completos del sistema estarán disponibles para todo el mundo, sin tener que pagar por un programa. Como resultado, un usuario que necesita cambios en el sistema será siempre libre para hacerlos por sí mismo.
